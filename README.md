# Tripsetta Ticket Form

A single-page web form for the team to file dev tickets into the Tripsetta
**Developer Items** Notion board. You describe a bug or idea in plain words;
Claude turns it into a clean title + description and tags it (Type / Platform /
Workflow) before creating the Notion page.

**Live form:** https://<your-username>.github.io/tripsetta-ticket-form/
